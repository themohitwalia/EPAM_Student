# EPAM Student Python Practice Repository

This repository contains a collection of Python exercises and practice tasks completed as part of the EPAM student training program. The main goal of these tasks was to build a strong foundation in Python programming by working on structured assignments that focus on core programming concepts and problem-solving techniques.

The exercises helped me improve my ability to write clean, readable, and well-structured Python code while applying basic software development practices.

---

## Overview

During the training program, I worked on multiple tasks designed to cover important Python concepts such as data structures, functions, decorators, recursion, and object-oriented programming. Each task focuses on a specific topic and helps develop practical coding skills.

This repository represents my learning progress and demonstrates my ability to implement Python concepts through hands-on coding exercises.

---

## Key Topics Covered

### Python Fundamentals
- Working with numbers and mathematical operations  
- String manipulation and formatting  
- Understanding variables and basic syntax  

### Data Structures
- Lists and list operations  
- Tuples and immutable data  
- Dictionaries and key-value data handling  

### Functions and Decorators
- Function arguments and return values  
- Writing reusable functions  
- Implementing and understanding Python decorators  

### Recursion
- Solving problems using recursive approaches  
- Understanding recursive function flow  

### Object-Oriented Programming
- Creating classes and objects  
- Encapsulation and abstraction  
- Class methods and instance methods  

### Magic Methods
- Implementing Python special methods such as `__str__`, `__repr__`, and others  
- Understanding how Python handles object behavior internally  

### Exception Handling
- Managing errors using try/except blocks  
- Writing robust and fault-tolerant code  

### Basic Web Scraping
- Introductory practice using a scraper template  
- Understanding how data can be extracted from web sources
